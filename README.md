# LandDrop
This app allows you to quickly and easily transfer files and documents from your phone or any device that supports QR codes and the web to your computer, using local network only, without the need for cables or external internet.
--------------------------------------------------------
هذا التطبيق يتيح لك نقل الملفات والمستندات بسرعة وسهولة من الهاتف أو أي جهاز يدعم QR والويب إلى الكمبيوتر، باستخدام الشبكة المحلية فقط، دون الحاجة لكابلات أو إنترنت خارجي.
